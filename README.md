Project Overview
This is an interactive 3D Christmas Tree experience created by Siheng Tao. The core code of this project was primarily generated by Gemini AI, with visual beautification and partial code adjustments made by the author.

Installation & Usage
1. Clone the Repo Download the source code to your machine:

Bash

git clone https://github.com/SihengTao/ChristmasTree_withGemini.git
cd ChristmasTree_withGemini
2. Install Dependencies Download necessary libraries (Three.js, React, MediaPipe, etc.):

Bash

npm install
3. Run Locally Start the development server:

Bash

npm run dev
Then open your browser at http://localhost:5173/.
